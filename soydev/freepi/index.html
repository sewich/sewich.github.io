I disassembled some crappy broken lenovo laptop recently and left the motherboard because it was a small rectangle and reminded me of SBCs. I wanted to see how much i can do without cooling and i could boot debian with KDE Plasma and play minesweeper, palapeli and chess. See? Told you gayming on linux is not a problem. All i can do for a case is screw it to a plank and add walls, and make a wooden case like for an electric contraption for school project. I could add a heatsink but i don’t want thermal paste to be the only thing keeping it glued to the CPU. It would be pretty useful to have some cooling because there is no CPU clock speed settings to prevent overheating, and it overheats when playing a youtube video, which shuts it down. Yes, you can just leave the fan for cooling and still use it but that would be boring, and i don’t want to actually use this crap for work, and it would just make the same annoying noise.
